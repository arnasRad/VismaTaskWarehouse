System was built with Intellij IDEA 2019.2.4 (Ultimate Edition)
java version 11.0.2
SDK 11

Project GitHub link: https://github.com/arnasRad/VismaTaskWarehouse.git

To launch the program, either open .exe file located in "Warehouse" folder, 
	or build the source code in the project located in "VismaWarehouse"
Note that data files needs to be in "Warehouse" folder for warehouse.exe to
	locate it and in "VismaWarehouse" folder (root project path) when
	building the project source code.

Further instructions on how to use the program are described in application
	"ABOUT" section.
